{ Interaction == Awareness }

For Ascentium Capital, I was contracted to create an interactive informational campaign, educating others on the benefits of Section 179 and how it can be utilized.

This project was built using pure HTML5, and was injected into an iframe hosted on Ascentium Capital's website, bringing awareness to the campaign. Adding an interactive twist, Ascentium tasked me to turn a boring survey into a smooth, interactive user experience.

Frontend

This web application was built with raw HTML and JQuery. No external rendering engines were used, and the entire gamified quiz was built to a spec.

Sharing

The client wanted to entice the visitor to share the gamified campaign to their friends via social media. A completion screen featured 3 social media sharing buttons: Facebook, Twitter, & LinkedIn. When sharing, the user is given a pre-written message based on their survey results.

Call-to-Action

The client wanted to bring impressions to their Section 179 Calculator, hosted on their website. To implement this, a completion screen featured a quick message, attracting the user to click a CTA button leading to their internal tool.

Adapting to screens

Responsiveness

Responsive design practices were used to ensure a seamless experience on any screen size. Focusing on responsiveness first ensures a smooth transition across devices. For Ascentium, this was an integral part of their iframe experience.

Gamification === Awareness

For Ascentium Capital, I was tasked with creating an interactive informational campaign, educating others on the benefits of Section 179 and how it can be utilized.

This project was built using pure HTML5, and was injected into an iframe hosted on Ascentium Capital's website, bringing awareness to the campaign. Adding an interactive twist, Ascentium tasked me to turn a boring survey into a smooth, interactive user experience.

{ Gamification == Education }

For Qeexo, I was contracted to create an iPad game for the youth demographic. The client requested development in LibGDX, with deployment on iOS devices using RoboVM.

A fun, simple user experience would allow Qeexo to use the game to collect telemetry data on the children, giving them insight to help define a personalized educational path.

Putting Java on iOS

LibGDX, a Java library based on LWJGL, can be combined with RoboVM, to develop deployments on iOS devices. This integration granted a wide toolbox to efficiently develop the application.

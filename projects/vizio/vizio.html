2020 Voice Experience/SystemUI

Developed an overlaying application, SystemUI, integrating the UI Component Library, building the Voice Search experience, facing the user for all voice search capabilities. Included multiple custom UI domains, such as "Show me the weather," "What is the Amazon Forest," all displaying unique modals showcasing the response to the user.
